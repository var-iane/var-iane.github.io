The palette of experimental fruits

Fruits are colorful and they taste good. But can we quantify these traits? This is an attempt to do so with a variety of unusual data. For example, I scraped patents of ~100 pluot varieties for skin and flesh color (reported with the Munsell color system), then converted those to hex codes to render a spectrum of fruit color.

Python (pandas, BeautifulSoup) for web scraping, data cleaning, data analysis
Datawrapper for data visualization
